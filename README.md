# Luigi's Pizza

A React-based web application for exploring and ordering from Luigi's Pizza menu. 

## Features
- **Responsive Navigation:** Easily browse through the website with a seamless navigation bar.
- **Hero Section:** A welcoming landing area with high-quality pizza imagery to engage customers.
- **Interactive Menu:** Browse a variety of delicious pizzas and beverages.
- **Contact Page:** A dedicated contact section for inquiries and orders.

## Technologies Used
- React.js
- React Router DOM
- CSS

## Getting Started

### Prerequisites
Make sure you have Node.js and npm installed on your machine.

### Installation
1. Clone the repository.
2. Navigate to the project directory:
   ```bash
   cd project01
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
To start the development server, run:
```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) to view it in your browser. The page will reload when you make changes.

## Project Structure
- `src/Components`: Contains reusable React components like `Navbar`, `Hero`, `Menu`, `Contact`, and `Footer`.
- `public`: Contains static assets like the `index.html`.
- `src/App.js`: The main application layout and routing setup.
